# SEO-Awesome 出海SEO自动化

一手数据驱动，零第三方工具费，全链路自动化。

## 这个技能干什么

用Google官方API拿一手数据，AI组装内容，模板渲染页面，自动发布，GA4自动复盘。从选词到上线到变现到迭代，搭一个自己会跑的SEO系统。

不教你"怎么做SEO"，教你"怎么搭一个自动做SEO的系统"。

## 核心理念

- 一手数据大于一切。Google API直连，不用Ahrefs、Semrush等第三方工具
- 数据自己会说话。GA4数据自动回流，AI按数据执行判断，人只定方向
- 页面是API数据组装的资产，不是AI凭空写的内容
- 变现从第一天就想。affiliate优先，AdSense兜底
